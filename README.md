# Facilities Management System
This is a backend project to manage facilities.

# Tech Stack
 - Spring boot
 - Spring data JPA
 - Postgresql
 - Liquibase
 - REST API
 - Kafka

# Prerequisites
 - Postgresql at `localhost:5432` having a database named `fms`
 - A Kafka bootstrap server at `localhost:9092`
 - Kafka topics (`email`, )
